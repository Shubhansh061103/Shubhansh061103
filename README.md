# 🚀 AI Doctor Appointment & Health Assistant

> An intelligent healthcare platform that simplifies doctor booking and provides AI-powered health assistance.

---

## 🌟 Overview

This project is a **full-stack AI-powered healthcare system** that allows users to:

- 🏥 Book doctor appointments seamlessly  
- 🤖 Get AI-based health suggestions  
- 📊 Track medical history & reports  
- 💬 Interact with an intelligent health assistant  

---

## ✨ Features

### 👤 User Features
- Secure authentication (JWT-based)
- Book, reschedule, and cancel appointments
- View doctor availability in real-time
- Access medical history and reports

### 🤖 AI Health Assistant
- Symptom-based suggestions
- Basic diagnosis guidance (non-clinical)
- Personalized health recommendations
- Chat-based interaction

### 🩺 Doctor Panel
- Manage appointments
- View patient history
- Update availability slots

### 📊 Admin Dashboard
- User & doctor management
- Analytics and monitoring
- Role-based access control
